ずっと気になっていたウルトラワイドモニターをついに購入しました。
選んだのはMSIの**Modern MD342CQP**です。
購入してしばらく使ってみたので、感想をまとめていきます。

## なぜウルトラワイドにしたのか

以前は普通の27インチモニターを使っていたのですが、コードを書きながらブラウザやターミナルを並べると、どうしてもウィンドウが窮屈になってしまっていました。
デュアルモニターも検討しましたが、ベゼルが視界に入るのが気になって、それなら一枚のワイドモニターにしようと考えました。

また、在宅作業が増えてきたこともあって、せっかくなら作業環境をちゃんと整えようと重い腰を上げた次第です。

## MSI MD342CQPのスペック

購入したのはMSIの**Modern MD342CQP**です。主なスペックは以下の通りです。

| 項目 | スペック |
|------|---------|
| パネルサイズ | 34インチ |
| 解像度 | 3440 x 1440（UWQHD） |
| アスペクト比 | 21:9 |
| パネル種類 | VA |
| 湾曲 | 1500R |
| リフレッシュレート | 120Hz |
| 応答速度 | 1ms（MPRT） / 4ms（GTG） |
| 輝度 | 300 cd/m² |
| コントラスト比 | 3500:1 |
| USB-C給電 | 98W |

接続端子は**HDMI 2.0b × 2、DisplayPort 1.4a × 1、USB-C × 1**と、主要な入力はひと通り揃っています。
USB-C一本でノートPCと映像出力・98W給電を同時にできるのは非常に便利です。

KVMスイッチ機能も内蔵しているので、複数のPCをキーボード・マウス共有しながら使えるのも地味に嬉しい点でした。

## 使ってみた感想

### 良かった点

**画面の広さは想像以上**でした。
3440 × 1440の解像度で34インチなので、コードエディタとブラウザとターミナルを横並びにしてもそれぞれ十分なスペースが確保できます。
ウィンドウを切り替える回数が明らかに減って、作業のテンポが上がりました。

**VAパネルのコントラストも印象的**で、暗いシーンや濃い色の表示がきれいです。
コントラスト比3500:1は伊達じゃないなと感じました。

**スタンドの使い勝手が思ったより良かった**のも意外なポイントです。
高さ調整（0〜110mm）、スイベル（±30°）、チルト（-5°〜+20°）と一通り揃っていて、自分に合った角度にすぐ調整できました。

### 気になった点

**VAパネルなので視野角はIPSに比べると若干狭め**です。
真正面から見ている分には問題ないのですが、斜めから見ると色が少し変わって見えます。
一人で正面から使う分にはまったく気にならないレベルですが、複数人で画面を共有する場面では少し注意が必要かもしれません。

**1500Rという湾曲**は慣れるまで少し時間がかかりました。
最初は「曲がってるな〜」と気になりましたが、数日で完全に慣れてしまいました。
むしろ今は視野に自然と画面が入ってくる感じで、没入感があって好きです。

**解像度が高い分、GPUへの負荷は増えます。**
3440 × 1440はFull HDの約2.5倍のピクセル数なので、ゲームをしない自分でも気にしておきたい点でした（主にコーディングや動画視聴なので実用上は問題なし）。

## まとめ

- 作業スペースが広がって生産性が上がった
- USB-C 98W給電でケーブルがスッキリした
- VAパネルのコントラストが思ったより良い
- 湾曲は数日で慣れる

デュアルモニターかウルトラワイドか迷っている方には、ウルトラワイドをおすすめしたいです。
ベゼルが視界に入らず画面が一体感を持って広がるのは、使ってみないとわからない快適さでした。

テレワークの作業環境を整えたい方は、ぜひ検討してみてください！
